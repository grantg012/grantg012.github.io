  • Timezone Adjuster

Pass a date in the URL query (`?11:30:00EST`) to have it rendered in the local timezone.
Note that valid timezones are Eastern, Central, Mountain, and Pacific Standard Time (xST) and Daylight Time (xDT). In this implementation, xST will never use daylight savings and daylight time will only use it when active.
